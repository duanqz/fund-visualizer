# Fund Visualizer 

数据展示基于Angular 6、Bootstrap 4和[Nebular](https://github.com/akveo/nebular)实现，是一个部署在github上的前端页面。

点击访问<https://fundvis.nullpointer.ltd/>，主要展示以下三部分信息：

- **持仓推荐**：对于爬取的基金排行数据，会根据一年、两年、三年、成立以来、三个月、五个月的收益率进行排序，找出位于头部的基金
- **基金排行**：展示不同类别基金的排行榜
- **基金详情**：目前仅展示基金的历史净值图表
